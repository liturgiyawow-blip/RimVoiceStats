
## 📦 RimMind Personality Web — Снапшот разработки v2.1
**Дата:** 2026-07-04  
**Статус:** Работает. Остались мелкие доработки UI.

---

### 1. Что это

Модуль `personality_web` для прокси-сервера RimMind. Добавляет веб-интерфейс и API для генерации психологических профилей колонистов RimWorld через локальную LLM (LM Studio).

**Функции:**
- Перехват данных пешек из промптов игры
- Генерация: описание личности, рабочие/социальные тенденции, «чистый профиль», история дня
- Озвучка сгенерированных историй через Piper TTS
- Веб-интерфейс: сетка пешек + отдельная страница профиля

### 3. Ключевые технические решения

| Компонент | Решение |
|-----------|---------|
| **Перехват данных** | `intercept_pawn_data()` в `routes.py` — парсит `[Name's Status]` из сообщений к LLM, вызывается в `voice_api.py` → `proxy_chat()` |
| **Кэш пешек** | `PawnCache` (JSON-файл `personality_pawn_cache.json`), потокобезопасный (RLock) |
| **Генерация профилей** | `PersonalityLLMClient` — 4 метода: `generate_field`, `generate_all`, `generate_pure_profile`, `generate_daily_story` |
| **Парсинг LLM** | Многоуровневый: `json.loads` → markdown-strip → regex-fallback. Обрабатывает «грязный» JSON с переносами строк |
| **Озвучка** | Через существующий `/rimmind/voice/speak` (Piper TTS), голос берётся из профиля или дефолт |
| **UI** | Чистый HTML+CSS+JS, без фреймворков. Модальные окна для длинного текста, diff-обновление без мигания |

### 5. Известные проблемы и фиксы

| Проблема | Причина | Фикс |
|----------|---------|------|
| Дублирование роутов | `register_personality_routes` вызывался 2 раза | Guard `if app.config.get('_PERSONALITY_ROUTES_REGISTERED')` |
| Пешки не появлялись | `intercept_pawn_data` не вызывался | Добавлен вызов в `voice_api.py` → `proxy_chat()` |
| Обрезка текста «Сгенерировать всё» | `ALL_MAX_TOKENS = 500` | Поднято до 1500 |
| Обрезка истории дня | `STORY_MAX_TOKENS = 600` | Поднято до 900 |
| JSON с переносами строк | LLM вставляет `\n` после `"story": "` | Улучшен regex в `_extract_field` |
| Прокрутка в модалке | `display: flex` на `.modal` ломал `overflow` | Убран flex, `overflow: hidden` на `.modal`, `overflow-y: auto` на `.modal-body` |
| Мигание страницы | `setInterval(loadPawns, 5000)` перерисовывал всё | Убрано автообновление, ручное + diff-проверка |

### 7. Быстрый старт для новой команды

1. Убедиться, что `personality_web/` лежит рядом с `main_proxy.py`
2. Проверить `personality_web/templates/` — должны быть `main.html`, `pawn.html`, `error.html`
3. В `main_proxy.py` **не** вызывать `register_personality_routes` — он вызывается внутри `voice_api.py`
4. Запустить прокси, открыть окно пешки в игре, зайти на `http://127.0.0.1:1235/rimmind/personality`
