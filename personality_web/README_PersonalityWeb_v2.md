# RimMind Personality Web — Расширение v2.0

## Что нового

### 1. Модальное окно персонажа (клик по плашке)
- При клике на карточку пешки открывается **полноценное модальное окно** со всеми данными
- Паспорт: раса, пол, возраст, настроение, идеология, черты, генетика
- Состояние здоровья и гены
- Топ навыков с цветовой индикацией (красный >10, жёлтый >7)
- **Воспоминания (Memory Module)** — полный список событий из `memories` с обратной сортировкой
- Связи с другими колонистами
- Быстрые кнопки на плашке: «Подробно», «✨ Профиль», «🔮 История»

### 2. «Чистый профиль» RimWorld (✨)
- Новый endpoint: `POST /rimmind/personality/generate_pure`
- LLM генерирует **лаконичный психологический портрет** в стилистике игры
- Без цифр, без игровых терминов — только атмосфера, характер, реакции
- Сохраняется в поле `pure_profile` в кэше

### 3. История дня / Story Mode (🔮)
- Новый endpoint: `POST /rimmind/personality/generate_story`
- LLM анализирует `memories` пешки и связывает события в **связную сюжетную цепочку**
- Учитывает других колонистов (при `include_colony: true`)
- Переводит игровые термины в человеческий язык
- Добавляет внутренние переживания: страх, надежду, раздражение, юмор
- Сохраняется в поле `daily_story`

### 4. Улучшенный UI
- Цветовые индикаторы статуса на плашках:
  - 🟢 Зелёный — есть классический профиль
  - 🔵 Синий — есть чистый профиль
  - 🩷 Розовый — есть история дня
  - 🟡 Жёлтый — пустая плашка
- Цветовые теги: черты, настроение, здоровье (красные), гены
- Анимации, hover-эффекты, липкий сайдбар
- Закрытие модалки по Escape или клику на оверлей

## API Endpoints

| Метод | URL | Описание |
|-------|-----|----------|
| GET | `/rimmind/personality` | Веб-интерфейс |
| GET | `/rimmind/personality/pawns` | Список всех пешек |
| GET | `/rimmind/personality/pawns/<name>` | Одна пешка (JSON) |
| POST | `/rimmind/personality/generate` | Генерация одного поля |
| POST | `/rimmind/personality/generate_all` | Генерация всего профиля |
| POST | `/rimmind/personality/generate_pure` | **Чистый профиль RimWorld** |
| POST | `/rimmind/personality/generate_story` | **Сюжетная история дня** |
| POST | `/rimmind/personality/clear` | Очистить кэш |

## Установка

1. Замените `personality_web/llm_client.py` на новый файл
2. Замените `personality_web/routes.py` на новый файл
3. `__init__.py` менять **не нужно** — импорт остался прежним
4. Перезапустите прокси

## Интеграция с RimMind Mod

Модуль `memory` RimMind уже используется через поле `memories` в кэше пешек.
Данные автоматически собираются при открытии окна персонажа в игре.
Для истории дня используются последние 8 записей `memories` ≈ один игровой день.
